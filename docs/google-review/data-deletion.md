# Data Deletion Policy

**Last Updated**: July 1, 2026

At **youtubeplaylistfinder.com**, we respect user rights to check, correct, and delete personal data in compliance with GDPR, CCPA, and Google API Policies.

---

## 1. Local Browser Storage
All bookmarks, checked videos, and learning streak metrics are stored in the client-side browser storage (`localStorage`). 
- **Wiping Data**: Users can erase this data instantly by clicking the "Clear Local Progress & Storage" button on our `/delete-data` page, or by clearing their browser's site cookies and settings cache.

---

## 2. Server logs and Caching
Search queries and public metadata fetched from YouTube are cached temporarily to prevent API quota issues.
- **Requesting Deletion**: Users may request complete erasure of IP addresses and query logs linked to their usage.
- **Submit Request**: Send a request via email to `youtubeplaylistfinder100@gmail.com` with the subject line "Data Deletion Request".
