# Privacy Policy

**Last Updated**: July 1, 2026

At **youtubeplaylistfinder.com**, we prioritize the privacy and security of our visitors. This Privacy Policy details how we handle data accessed through YouTube API Services, browser local storage, and website logging.

---

## 1. YouTube API Services
- **YouTube API Services**: This application uses YouTube API Services. By using our website, you agree to be bound by the [YouTube Terms of Service](https://www.youtube.com/t/terms) and agree to the [Google Privacy Policy](https://policies.google.com/privacy).
- **Google API Policy**: We strictly comply with the [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy).
- **Affiliation Disclaimer**: This application is not affiliated with, endorsed by, sponsored by, or certified by Google LLC or YouTube.

---

## 2. Data Collection
We only collect or process data necessary to provide search features:
- **Search Queries**: Used to search for public playlists on YouTube.
- **Playlist & Video IDs**: Used to resolve playlist outlines.
- **Device & Browser Details**: Anonymized viewport sizes and user agents.
- **Cookies & Analytics**: Standard analytics to track site performance.

**Privacy Guarantees**:
- We never collect YouTube or Google credentials/passwords.
- We never access or store private YouTube content.
- We never sell user data.
- We access only publicly available YouTube metadata.

---

## 3. Data Retention & Deletion
- Cache layers (KV Store) temporarily store query results for 24 hours to prevent quota abuse.
- All bookmarks and checkbox progress parameters are saved locally in the browser's `localStorage` and never touch our servers.
- Users can clear local data via browser settings or request log erasure via the `/delete-data` page.
